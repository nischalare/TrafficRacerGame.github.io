
# 🏎️ Traffic Racer – Browser Car Racing Game

This project is a **simple browser-based traffic racing game** built using **HTML, CSS, and JavaScript**.

Players control a car and try to avoid oncoming traffic while driving as long as possible.  
The game demonstrates basic **web game development concepts** such as DOM manipulation, animations, keyboard input handling, and browser rendering.

The project runs completely in the browser and requires **no installation or frameworks**.

---

# 🚀 Features

- 🏎️ Simple and fun car racing gameplay
- 🎮 Keyboard controls for car movement
- 🚗 Incoming traffic obstacles
- ⚡ Increasing challenge as the game progresses
- 🌐 Runs directly in the browser
- 📱 Lightweight and fast loading

# 🎮 How the Game Works

1. The game loads in the browser using HTML and CSS for layout.
2. JavaScript controls the **car movement and obstacle generation**.
3. The player moves the car using **keyboard arrow keys**.
4. Traffic cars appear randomly and move toward the player.
5. If the player collides with another car, the **game ends**.

The objective is to **survive as long as possible without crashing**.
